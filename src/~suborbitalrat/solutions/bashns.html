BashNS

BashNS is a set of tools I've created for the online game NationStates to automate some of the gameplay using the backend API and the frontend html website.

For more info, see the git repo.

Why?

Because I'm a bored teenager with unrestricted access to the internet and I enjoy automating the hell out of things. In Bash, of course, because I'm an idiot.

Some warnings

The parts of BashNS that use the NationStates frontend site may be against the NationStates TOS and using them could get you banned from the game. I take no responsibility for your actions.
